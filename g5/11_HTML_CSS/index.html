Color

The color & background-color properties

  • The color property allows you to specify the color of text inside an element.
  • The background-color property sets the color of the background for the element box.

*You can specify any color in CSS in one of 3 classic ways:

  • color names
    • There are 147 predefined color names that are recognized by browsers. *For example: DarkCyan
  • rgb values
    • These express colors in terms of how much
      red, green and blue are used to make it up. *For example: rgb(100,100,90)
  • hex codes
    • These are six-digit codes that represent the amount of red, green and blue in a color, preceded by a pound or hash # sign. *For example: #ee3e80 


/* color name */
h1 {
	color: DarkCyan;
}
/* rgb value */
p {
	color: rgb(100,100,90);
}
/* hex code */
h2 {
	color: #ee3e80;
}

The opacity & rgba properties

  • CSS3 introduces the opacity property which allows you to specify the transparency of an element and any of its child elements.
  • The value is a number between 0.0 and 1.0 *(so a value of 0.5 is 50% opacity and 0.15 is 15% opacity)
  • The CSS3 rgba allows you to specify a color, just like you would with an RGB value, but adds a fourth value to indicate opacity / transparency.


/* opacity - defines transparency on both content and background */
h1{
	opacity: 0.5;
}

/* rgba value - defines transparency based on where you use it */
p {
	color: rgba(100,100,90,0.5);
}

CSS3: HSL & HSLA Colors

  • CSS3 introduces an entirely new and intuitive way to specify colors using hue, saturation, and lightness values.

*HSL - Hue, Saturation, Lightness


/* hsl */
body {
	background-color: hsl(0,0%,78%);
}

/* hsla */
p {
	background-color: hsla(0,100%,100%,0.5);
}
